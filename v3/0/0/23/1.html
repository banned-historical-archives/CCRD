中央办公厅关于对林彪、江青反革命集团主犯江青、张春桥将依法予以减刑的通知
1983.01.03；中办发[1983]3号



最高人民法院将对林彪、江青反革命集团主犯江青、张春桥依法予以减刑，现根据中央的指示，将有关情况通知如下：

(一)一九八一年一月二十五日，最高人民法院特别法庭对林彪、江青反革命集团的主犯江青、张春桥，判处死刑，缓期二年执行。至一九八三年一月二十五日，缓期二年执行期限届满。经中央政治局讨论，鉴于林彪、江青反革命集团造成的十年内乱早已结束，党的十一届三中全会以来，国家安定团结的政治局面日益巩固，当前形势很好，罪犯江青、张春桥继续危害国家和人民的可能已经排除；同时，江、张两犯在死刑缓期执行期间，遵守监规，无抗拒改造的恶劣情节，依照《中华人民共和国刑法》第四十六条关于对判处死刑缓期执行的罪犯，在死刑缓期执行期间，如果抗拒改造情节恶劣，查证属实才执行死刑的规定，最高人民法院将裁定对原判处江青、张春桥的死刑缓期二年执行的刑罚，减为无期徒刑。

最高人民法院裁定减刑后，由新华社发布消息，公布裁定书。

(二)对林彪、江青反革命集团主犯江青、张春桥依法予以减刑，我们相信绝大多数干部和群众是可以理解的。当然，在干部和群众中也会引起一些议论。请各级党委及时从政治上和法律上做好解释工作。党内外的重要反映，请随时上报中央。

(三)各地对林彪、江青反革命集团案的其他罪犯，必须继续抓紧审理，尽快依法作出判决，绝不能由于对江青、张春桥减刑而受到任何影响。要警惕林彪、江青反革命集团残余势力和其他敌对分子乘机进行捣乱破坏，凡是进行现行反革命破坏活动查有实据的，要坚决依法予以打击。



(原载《干审工作政策文件选编》，北京：党建读物出版社，1993年)
