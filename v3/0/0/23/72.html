中央办公厅转发中央统战部《关于当前落实政策的情况和今后工作意见的报告》
1983.08.16


各省、市、自治区党委，各大军区、省军区、野战军党委，中央各部委，国家机关各部委党组，军委各总部、各军兵种党委，各人民团体党组：

中央统战部《关于当前落实政策的情况和今后工作意见的报告》，已经中央书记处同意。现转发给你们，请参照执行。

中央书记处指出：落实各项统战政策，是党的十一届三中全会以来拨乱反正的重要内容之一，是关系到端正党风、恢复党的优良传统的大事。几年来，各级党委在这方面做了大量的工作，成绩是显著的。但是，必须明确指出，现在不是落实政策过了头，而是差距很大。各级党委要继续清除“左”的思想影响，采取有力措施，把这项工作切实抓紧抓好。在落实政策工作中，对于顶着不办的，党委的负责同志要直接干预，限期解决。要坚决反对无组织无纪律的行为和官僚主义的作风。

请各省、市、自治区党委于今年年底前将执行情况向中央写一专题报告。

 中共中央办公厅
一九八三年八月十六日

(此件发至县、团级)



关于当前落实政策的情况和今后工作意见的报告


中央书记处：

中央统战部趁各地统战部部长来京参加六届人大、政协一次会议之便，召开了全国统战部部长座谈会。会议主要就抓紧落实各项统战政策问题，学习讨论了胡耀邦同志在人大、政协会议期间关于落实政策的几次批示，交流了各地落实政策情况，研究了今后的工作。会议一致认为，几年来落实政策工作，虽然取得不少成绩，但问题还很多。胡耀邦同志一九八二年一月在全国统战工作会议上提出的落实政策的“七年”之期，到今年年底，行将期满，在今后几个月中，务要加倍努力，抓出显著成绩。

一、几年来落实政策的进展情况

十一届三中全会以来，在党中央的领导下，由于各级党委、统战部门和有关方面的共同努力，落实统战政策的工作取得了很大的进展。对被错划为右派的已经改正，对他们的工作和生活也做了妥善安置。原工商业者中的冤假错案基本上已平反纠正，在“文革”中被查抄的巨额存款已经退还(全国约退还四亿元)，被扣减的工资绝大部分已补发(全国约补发三亿三千万元)，被撤销职务的大部分已经恢复或作了相应安排，他们的生活待遇问题也按有关规定得到了解决。原为劳动者的小商小贩、手工业者已经从原资产阶级工商业者中被区别出来。对民主党泒成员、非党知识分子，复查平反了大量寃假错案，改善了一部分人的工作条件和生活待遇。“文革”中被占的民主党派、工商联办公用房，也已退还了一部分。原国民党起义人员，已复查平反寃假错案十一万一千多件，对其中需要安排工作的六万八千人作了安置，对一批生活困难的进行了救济，对在起义中有重大贡献或有较大代表性的人物，在各级政协和政府中作了适当安排。全国台属需要落实政策的有九万三千余件，已落实八万余件。居住在祖国大陆上的台胞中寃假错案已大部平反，相当一部分家庭的子女就业、住房困难、生活不到中等水平问题已得到解决。对少数民族中的寃假错案进行了认真复查，民族地区的大案、要案已基本上解决。落实宗教政策方面也做了大量工作，取得了很大成绩。

随着看各项统战政策的落实，妥善地解决了长期遗留的许多问题，有力地调动了广大知识分子、民主党派成员、原工商业者、台胞、台属、起义人员、少数民族上层人物和宗教界人士为社会主义服务的积极性，许多人在各自的工作中做出了显著成绩，有些人被评为先进工作者。事实再次证明，中央关于坚决落实各项统战政策的方针是完全正确的，对于加强和巩固爱国统一战线，团结广大党外人士努力实现八十年代三大任务，具有重大的作用和深远的影响。

二、存在的主要问题

落实政策的工作虽然取得了显著成绩，但我们一定要保持清醒的头脑。正如胡耀邦同志最近指出的：“现在的问题是对知识分子、民主人士、起义人员和归侨等方面的政策落实不是过头了，而是差距很大。”

当前存在的主要问题是：

(一)工作进展很不平衡。有些省落实台属政策的任务完成还不到二分之一。有的省落实政策工作没有领导人具体负责，工作停滞不前。有的地方一些负责人还顶着不办，使许多问题久拖不得解决。如福建永定县对一起义人员落实政策拖了两年，省里督促达一、二十次，县委书记就是顶着不办。浙江慈溪县有一个原国民党自卫队，经调查是起义部队，有关部门上报金华市，因市委个别领导人思想不通，压了一年多，至今还未解决。类似这种问题，其他地区也存在。

(二)遗留问题较多，特别是“文革”中大量被占用的房屋和查抄的财物还未归还原主。被长期占用至今未退的有私人房产，也有民主党派、工商联的办公用房和宗教团体的房产。这类问题在北京、上海、江苏、浙江、四川、辽宁、山西、安徽等许多地区普遍存在。如上海市一些有影响的代表人物的住房也未退还。江苏省工商联和十一个县、市工商联，“文革”前办公用房有四百七十三间，现在仅有七十八间，其余房屋均未退还。重庆市不少区民建、工商联的办公用房被区委统战部和区政协占用。南通市民主党派和工商联办公用房也被长期占用，各党派已向法院起诉。宗教团体的房屋同样被大量占用，现在全国开放的寺庙教堂仅有一万四千余座，相当于“文革”前的一半，其中有一万二千座是伊斯兰教清真寺，而散居地区开放的很少，天主教、基督教的宗教场所开放的更少，引起信教群众的不满。在退还查抄的财物工作上，遗留问题更多，不少被查抄的财物没有下落，有的没有清单，有的单位不认真查找，在群众中造成很不好的影响。

(三)复查建国初期起义人员案件的任务还很重。按照一九八二年十一月中共中央办公厅《转发中央统战部等六部门〈关于解决落实起义、投诚人员政策工作中遗留问题的报告〉》(中办发[1982]41号文件)的精神，对建国初期审理的起义、投诚人员案件要进行普遍复查，工作量很大，特别是过去起义人员比较集中的地区更为突出。如湖南省一万九千件，现在只复查了七千五百件，尚有一万二千件待查。

存在上述问题的原因：一是“左”的思想影响尚未清除。有的认为落实政策是“右倾”，认为落实台属政策是“为国民党效劳”、“统战部是国民党的服务部”，有的县委书记对落实起义人员政策思想不通，说“这些都是反革命，现在变成革命了，要慎重”，甚至有的省委的领导同志也有这种错误认识。二是党风不正。有的党员，甚至相当负责的干部，无视党的组织纪律，不执行党的政策。三是对落实政策的重要性认识不足。有的省在机构改革时，把有关落实政策小组领导成员全部调离工作岗位，使工作陷于停顿。四是存在实际困难。如对建国初期案件的复查，由于年代久，人事变动，又缺乏档案资料，哪些人是起义人员一时确难分清。有些占用房屋的单位房源有困难。

三、对今后工作的意见

上述情况说明，落实政策面临的任务还非常艰巨。我们一定要下决心，采取有力措施，抓紧督促检查，争取在今年年底以前做出显著成绩。

(一)加强思想教育，严肃党纪。落实党的统一战线政策，是顺利实现四个现代化，巩固和发展祖国大团结大统一的需要，是开创统一战线工作新局面的关键，是检验我们同党中央在政治上是否保持一致的重要标志。对于这个重大原则问题，决不能含糊敷衍。要坚决消除“左”的影响，把思想教育和执行纪律结合起来，以坚定的态度把这项工作做好。对拒不执行政策、顶着不办的，一定要严肃处理。

(二)认真进行督促检查。中央统战部今年下半年要派干部到一些地区进行督促检查。各省、市、自治区也要派人下去督促检查。督促检查要采取党内外相结合的办法，请政协、各民主党派和工商联的人士一起参加，听取他们的意见，发挥他们的作用。落实政策工作基本结束的地区，要认真验收，作出总结，工作一定要扎实，要经受得住历史的检验。

(三)抓紧解决老大难问题。有些老案，建议中央各部门和各省、市、自治区党委要直接处理，不要层层照转，一定要一件一件地抓好落实。退还房屋问题，要分别情况处理：对于占用的私人房产，凡是现在能解决的就现在解决，能部分解决的就先部分解决，确有困难一时不能全部解决的，要订出计划，和本人讲清楚，争取两三年内基本解决。对于民主党派、工商联的房屋，凡统战部门占用的，今年十月一日前必须一律退还；其他机关团体占用的，也要在今年年底前全部退还。对于宗教团体的房屋，要按照一九八○年七月《国务院批转国务院宗教局、国家建委等单位关于落实宗教团体房产政策等问题的报告》(国发[1980]188号文件)的规定和一九八二年三月《中共中央关于印发〈关于我国社会主义时期宗教问题的基本观点和基本政策〉的通知》(中发[1982]19号文件)的精神办理。要严格执行今年四月《国务院批转国务院宗教事务局关于确定汉族地区佛道教全国重点寺观的报告的通知》(国发[1983]60号文件)。对于代管的房产，城乡建设环境保护部正拟订处理办法，待报中央和国务院批准后，按批准的规定执行。对于起义人员建国初期的案件，现在没有档案材料的，应以从宽从了的精神处理，不得拖压不办。对于查抄的财物，各地也要针对具体情况，提出措施，抓紧合理解决。

(四)各级党委要加强领导。落实政策是全党的工作，必须在党委统一领导下，各部门积极配合，全党来抓。今年下半年落实政策能否取得突出成绩，最重要的是取决于各级党委的重视程度，能否采取得力措施，一抓到底。现在各地正在调整机构，各级党委不要因此而放松对此项工作的领导，分管这一工作的书记、常委，应认真地把工作抓紧抓好。各地统战部门要经常向党委反映情况，请示报告，取得领导的支持。

以上报告，如无不当，请批转各地参照执行。

 中央统战部
一九八三年七月二十五日


(原载《公安建设》，1983年第15期，第24-29页。)
