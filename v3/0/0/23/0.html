文章全文：

中共中央对广西自治区党委关于进一步处理好广西“文化大革命”遗留问题的请示报告的批复
1983.01.02；中委会[1983]1号



中共广西壮族自治区党委：


你们《关于进一步处理好广西“文化大革命”遗留问题的请示报告》已收悉。

《报告》检查了你们在处理“文化大革命”遗留问题方面存在的缺点，这种精神很好，所提进一步处理好这个问题的意见，中央原则同意。

中央希望你们进一步以党的十一届六中全会一致通过的《关于建国以来党的若干历史问题的决议》统一自治区的全体党员同志和有关群众的认识，大力发扬党性，坚决克服派性，坚定不移地本着“实事求是，有错必纠”和“宜粗不宜细”的精神，抓紧平反一切冤假错案，尽快做好落实党的政策的工作。对情况复杂的“老大难”问题，各级党政主要领导同志要亲自动手，调查研究，做好工作。自治区党委对这个文件的贯彻执行情况，一个季度要检查一次，并向中央作出报告。

中央相信，只要你们从思想上引起重视，在工作上措施得力，就一定能够尽快地解决好“文化大革命”的遗留问题，进一步增强全区各族干部、群众的团结，把广西的各项工作做得更好。


 中共中央
一九八三年一月二日


来源： 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。
