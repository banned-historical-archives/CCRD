中共中央转发吉林省委“五反”简报上反映的边反边犯的几个事例材料的批示
1963.07.20


各中央局，各省、市、自治区党委，中央各部委、各党组：


现将吉林省委“五反”简报上反映的边反边犯的几个事例的材料，转发给你们一阅。

在“五反”运动期间，仍然发生请客送礼、多吃多占、特殊化走后门这类事件，很值得注意。如果这些单位和个人，已经进行过“三反洗澡”，其性质尤为严重，是明知故犯，要严肃对待，从严处理。既然发生了边反边犯的现象，也说明运动还搞得不深，还没有使干部受到深刻的教育和取得应有的教训。要以这些事例作为活的教材，反复教育干部，边犯就边反，进一步把“五反”运动搞深搞透。


 中 央
一九六三年七月二十日
（发至县团级）
