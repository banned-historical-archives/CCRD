文章全文：

中共中央、国务院、中央军委、中央文革关于湖南红旗军、红导弹等红卫兵组织到湖南省军区动手打人、抓人问题的指示
1967.01.20


军队负有战备任务和保卫文化大革命任务。不能允许外人冲入指挥、保密等系统，不许接管。红旗军是反动组织，必要时将其少数头头逮捕，可起分化瓦解作用。军区负责人，应挺身而出，对群众进行说服教育工作。

 中共中央 国 务 院
中央军委 中央文革
一九六七年一月二十日凌晨
